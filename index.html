Portfolio

Crash Course on Linear Regression: A Comparison Between Classical and Bayesian Statistics

I began this project in my free time. In it I discuss linear regression from two different perspectives: classical and Bayesian statistics. The document is not ready yet, but the code I wrote for the project, including a user-defined function that estimates the Bayesian regression, is available at GitHub.

A Semiparametirc Poisson Autoregressive Model Applied to Vehicle Theft in Bogotá

This is my final project for the Bayesian Inference course. It's based on the proposal made by Aldor-Noiman, Brown, Fox & Stine, which uses semiparametirc methods to study criminal events. I'm currently working on its enhancement to publish it. The original document can be downloaded here.

Constrained Markov Chains: An Application for Melody Generation

This is my degree project. It's based on the proposal made by Pachet, Roy & Barberi , which uses Markov chains to create monophonic melodies that fulfill certain restrictions. The original document (in Spanish) and the code can be downloaded here.
